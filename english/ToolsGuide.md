# Tools for Mod Developers

### 1. JSON Schema for Mods

The official JSON schema highlights errors, autocompletes keys, and shows hints
in editors with LSP support (VS Code, IntelliJ IDEA).

Add to the top of `content.json`:

```json
{
  "$schema": "https://ext.nulls.gg/mods/schema/schema.json"
}
```

- Source code: [github.com/nulls-mods-community/schema](https://github.com/nulls-mods-community/schema)

### 2. SC Editor

A Java application for viewing and editing SC, SC2, and SCTX files (2D textures
and animations). Can render objects to an image or video.

- Download: [github.com/danila-schelkov/sc-editor/releases/latest](https://github.com/danila-schelkov/sc-editor/releases/latest)
- Requires Java (JRE). Supports Windows, macOS, Linux.
