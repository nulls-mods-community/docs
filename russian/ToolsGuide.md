# Инструменты для мододелов

### 1. JSON-схема для модов

Официальная JSON-схема подсвечивает ошибки, автодополняет ключи и показывает подсказки
в редакторах с поддержкой LSP (VS Code, IntelliJ IDEA).

Добавьте в начало `content.json`:

```json
{
  "$schema": "https://ext.nulls.gg/mods/schema/schema.json"
}
```

- Исходный код: [github.com/nulls-mods-community/schema](https://github.com/nulls-mods-community/schema)

### 2. SC Editor

Java-приложение для просмотра и редактирования SC, SC2 и SCTX файлов (2D-текстуры
и анимации). Умеет рендерить объекты в изображение или видео.

- Скачать: [github.com/danila-schelkov/sc-editor/releases/latest](https://github.com/danila-schelkov/sc-editor/releases/latest)
- Требуется Java (JRE). Поддерживается Windows, macOS, Linux.
