# Спецификация модификаций Null's Brawl (v0 unstable)

## Основные понятия
- **Модификация**: специальный пакет, загружаемый загрузчиком модификаций, изначально представленный в виде директории.
- **ID модификации**: корректный UUID, уникальный для каждой модификации, строго совпадающий с названием директории.

## Структура директории
- Должна содержать файл **content.json** в формате JSON, содержащий объект [Modification](#modification).

## Алгоритм загрузки
1. **Порядок обработки**:
   - Сама модификация.
   - Её отдельные фичи, упорядоченные по **@priority** (от меньшего к большему); фичи с одинаковым приоритетом упорядочиваются лексикографически по ID.
2. **Патчинг таблиц**:
   - Модификации не могут напрямую изменять файлы таблиц (например, `skin_confs.csv`).
   - Изменения (дельты) должны быть описаны так, чтобы загрузчик мог «отпатчить» таблицы игры.
   - Цель: минимизировать конфликты и упростить обновления.

## Изменения (патчинг)
- **Представление**: описывается через объекты [TablePatch](#tablepatch) внутри объекта [TablePatches](#tablepatches).
- **Пример**:
  ```json
  {
    "club_piggy_levels": {
      "PiggyLevel_0_0": {
        "State": 6,
        "ShownLevelInCounter": 5
      }
    }
  }
  ```
- **Типы значений**: типы JSON должны совпадать с типами в заголовке таблицы (String, Int, Boolean). `null` очищает ячейку. Несовпадения типов могут быть автоматически исправлены (не гарантируется).
- **Массивы**:
  - Используются, когда один объект содержит несколько строк (например, несколько цветов).
  - **Y > X**: загрузчик добавляет новые CSV-строки.
  - **X > Y**: загрузчик записывает первые Y значений; остальные ячейки очищаются (пустые строки удаляются).
  - `[X, Y, Z, null]` обрабатывается как `[X, Y, Z]`.
  - Массив из одного элемента `[X]` обрабатывается как значение `X`.
- **Новые объекты**: если JSON-изменение ссылается на неопределённый объект, он создаётся в конце таблицы (порядок для нескольких объектов не определён).

## Справочник JSON-объектов

### Modification
Описывает модификацию.

| Ключ | Тип | Значение |
| :--- | :--- | :--- |
| **@title** | [IntlString](#intlstring) | Название модификации (обязательно) |
| **@description** | [IntlString](#intlstring) | Описание модификации (обязательно) |
| **@author** | [IntlString](#intlstring) | Автор |
| **@version** | [Version](#version) | Версия (по умолчанию: "1.0.0") |
| **@gv** | Integer | Совместимая версия игры (обязательно) |
| **@spec** | Integer | Версия спецификации |
| **@ilib** | Boolean | Опубликована в библиотеке (внутренний) |
| **@categories** | String[] | Категории для библиотеки (внутренний) |
| **@features** | [Features](#features) | Дополнительные переключаемые фичи |
| **@feature_groups** | [FeatureGroups](#featuregroups) | Группы фичей |
| **@root** | String | |
| **@patches** | String[] | Расширяет [TablePatches](#tablepatches) |

### Features / Feature
- **Features**: объект с произвольными ключами (ID фичи), отображающими на объекты [Feature](#feature).
- **Feature**:
  - **@name** (обязательно): [IntlString](#intlstring)
  - **@description**: [IntlString](#intlstring)
  - **@enabled**: Boolean (по умолчанию: true)
  - **@conflicts**: String[] (ID конфликтующих фичей)
  - **@priority**: Integer (по умолчанию: 0)
  - **@root**, **@patches**: расширяет [TablePatches](#tablepatches)

### FeatureGroups / FeatureGroup
- **FeatureGroups**: объект с произвольными ключами (ID группы), отображающими на объекты [FeatureGroup](#featuregroup).
- **FeatureGroup**:
  - **@name** (обязательно): [IntlString](#intlstring)
  - **@description**: [IntlString](#intlstring)
  - **@type**: String ("DEFAULT" или "RADIO_GROUP")
  - **@features**: String[] (включённые ID фичей)

### IntlString
Локализованная строка.
- Может быть простой строкой.
- Или словарём, где ключ = код языка (ISO 639, заглавными буквами, например EN, RU), значение = текст.
- Ключ **EN** должен присутствовать (резервный вариант).

### Version
Строка формата "X.Y.Z" (например, "1.0.0").

### TablePatches / TablePatch
- **TablePatches**: ключи объекта = имя файла таблицы без расширения (не может начинаться с @). Значения = [TablePatch](#tablepatch).
- **TablePatch**: ключи объекта = имя объекта в таблице. Значения = объект с именами колонок в качестве ключей и значениями ячеек в качестве значений.
