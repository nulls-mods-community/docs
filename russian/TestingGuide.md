# Тестирование модов без подписи

Для тестирования в процессе разработки существует специальный APK Null's Brawl,
который позволяет загружать неподписанные моды.

### Способ 1: Developer APK

Начиная с версии 65.165 существует отдельный APK для загрузки неподписанных модов.

Актуальная ссылка публикуется в анонсах Telegram-топика
[t.me/nb_mods](https://t.me/nb_mods) в разделе #Announcements.

> **Важно:** APK предназначен только для тестирования собственных модов.
> Установка чужих неподписанных модов запрещена и может привести к блокировке аккаунта.

### Способ 2: Утилита Nulls TestMods (от darkmean)

Утилита автоматизирует установку мода для тестирования.

1. Установите Nulls TestMods (ссылка в чате мододелов).
2. Предоставьте приложению доступ к файлам.
3. Откройте ваш `.zip`, `.json` или `.toml` через «Поделиться» → «Nulls TestMods».
4. Утилита создаст папку `NullsBrawlMods` и распакует мод с UUID.
5. В Null's Brawl добавьте папку `NullsBrawlMods` через настройки модов
   (шестерёнка → «Добавить папку»), если ещё не делали этого.
6. Включите мод и перезапустите игру.

> Утилита генерирует UUID автоматически из названия мода. Для точного контроля
> указывайте UUID вручную через структуру папок (Способ 3).

### Способ 3: Ручная установка через папки (только для Developer APK)

1. Создайте папку для модов, например `NullsBrawlMods`.
2. Внутри создайте подпапку с именем, точно совпадающим с UUID мода.
3. Распакуйте файлы мода (`content.json`, ресурсы) в эту папку.
4. В Developer APK: настройки модов → «Добавить папку» → выберите `NullsBrawlMods`.
5. Перезапустите игру — мод появится в списке.

После добавления папки новые моды достаточно класть в подпапки с UUID —
они появятся автоматически при следующем запуске.
