# Мета-информация в модах

В `content.json` есть три обязательных поля: `@title`, `@description`, `@author`.
Они поддерживают локализацию и HTML-форматирование.

### Обязательные поля

```json
{
  "@title": "Название мода",
  "@description": "Описание мода",
  "@author": "Автор"
}
```

### Локализация

Вместо строки можно передать объект с кодами языков (ISO 639, заглавными буквами).
Ключ `EN` обязателен, так как он используется, если язык пользователя не найден.

```json
{
  "@title": {
    "RU": "Супер мод",
    "EN": "Super mod",
    "TR": "Süper mod"
  },
  "@description": {
    "RU": "Добавляет крутые скины",
    "EN": "Adds cool skins"
  },
  "@author": "Ваш ник"
}
```

### HTML-форматирование

Поля `@title`, `@description`, `@author` поддерживают HTML-теги
(на основе `android.text.Html`):

- `<b>жирный</b>`, `<i>курсив</i>`, `<u>подчёркнутый</u>`
- `<s>` / `<del>` — зачёркнутый
- `<font color='#RRGGBB'>цветной текст</font>`
- `<a href='URL'>ссылка</a>`
- `<br>` — перенос строки
- `<ul><li>элемент списка</li></ul>`
- `<p>`, `<div>`, `<span>`, `<big>`, `<small>`, `<tt>`, `<code>`, `<sup>`, `<sub>`, `<h1>…<h6>`

```json
{
  "@title": "<b>Супер мод</b>",
  "@description": "<p>Описание:</p><ul><li><i>Крутая фича</i></li></ul><br>Ссылка: <a href='https://t.me/nb_mods'>Чат</a>",
  "@author": "<a href='https://t.me/danyanull'>данянул</a>"
}
```

> Не забывайте экранировать кавычки внутри JSON.

### Рекомендации

- Начинайте предложения с большой буквы, ставьте пробелы после знаков препинания.
- Избегайте транслитерации: пишите "default", не "дефолтный".
- Описание должно отвечать на вопросы: что изменится и как это повлияет на игру.
